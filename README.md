# 🧮 Java Simple & GUI Calculator

Welcome to the **Simple Calculator** project! This repository contains a fully functional calculator application developed in Java. It features two variants to demonstrate both terminal-based logic and interactive desktop application design.

Developed as **Task 1** during my **Java Development Internship**.

---

## ✨ Features


- **GUI Mode (`GuiCalculator.java`):** An interactive, modern desktop app built using **Java Swing**.
- **Core Operations:** Supports Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
- **Robust Logic:** Includes edge-case handling like preventing application crashes during division by zero.

---

## 🛠️ Technologies Used

- **Language:** Java (JDK 8 or higher)
- **GUI Framework:** Java Swing & AWT
- **IDE:** Visual Studio Code (VS Code)
